# 📸 Giada Photography - Portfolio Fotografico

Sito web vetrina e portfolio responsive progettato e sviluppato per una fotografa professionista. Il progetto si focalizza su un'interfaccia elegante, minimalista e d'impatto visivo, pensata per valorizzare gli scatti e ottimizzare l'esperienza utente su qualsiasi dispositivo (Desktop, Tablet e Mobile).

### 🛠️ Tecnologie Utilizzate

* **HTML5:** Strutturazione semantica dei contenuti.
* **CSS3:** Stile personalizzato, gestione dei font e layout responsive.
* **JavaScript (Vanilla):** Implementazione di feature interattive nella galleria.
* **Media Queries:** Ottimizzazione completa per dispositivi mobile.

---

### 🚀 Funzionalità Principali

* **Design Responsive:** Interfaccia fluida che si adatta perfettamente a schermi di ogni dimensione.
* **Galleria Fotografica Dinamica:** Visualizzazione ottimizzata delle immagini tramite URL per garantire un caricamento rapido delle anteprime.
* **Form di Contatto Predisposto:** Layout per la richiesta di preventivi o sessioni fotografiche.

---

### 📈 Sviluppi Futuri & Roadmap

Il progetto è attualmente in una fase di MVP (Minimum Viable Product) statico. I prossimi step di sviluppo previsti sono:
1. **Integrazione Backend:** Collegamento del form di contatto a un servizio di gestione delle richieste (es. Formspree, EmailJS o un backend custom in PHP/Laravel).
2. **Local Storage delle Immagini:** Ottimizzazione e caricamento degli scatti direttamente all'interno degli asset di progetto con compressione WebP per migliorare la SEO e i tempi di caricamento.
3. **Sezione Galleria Filtrabile:** Implementazione di filtri avanzati in JavaScript per dividere gli scatti per categoria (es. Ritratti, Paesaggi, Eventi).
